# OCR 인식률 재테스트 리포트 (2026-07-10)

> 깃에 기록된 실측 OCR 라인(8장)을 현재 파서로 재실행해 필드 인식률을 수치화한다.
> "before" = PP-OCRv5(온디바이스), "after" = Apple Vision + 값-형식 분류(현재).
> 정답(ground-truth) 보유분(R01~R04)은 자동 채점, 미보유분은 육안 검수용으로 인식값을 나열한다.

## 1. 요약 지표

| 지표 | PP-OCR (before) | Apple Vision + 값-형식 (after) |
|---|---|---|
| 필수필드 채움 (배달일자·상품명·배달장소, 8장×3=24) | 10/24 | **22/24** |
| 전체 비어있지 않은 필드 수(8장 합) | 20 | **68** |
| 정답 정확일치 (R01~R04, 31필드) | 14/31 (baseline) | **26/31** |

- 정답 정확도: **14/31 → 26/31** (45% → **84%**).
- 이번 세션 개선: 값-형식 스캐너(상품명·엄수/예식시간·근조 수령인) + 주소 정제 + 문서순 발주/배송 전화.

## 2. 수령인 회복(레이아웃 붕괴 대응)

Apple Vision 경로에서 "받는분/고인/상주 + 이름" 및 "이름+부친상" 형식으로 수령인 6/7 회복
(R04 유기열 포함). PP-OCR 경로는 0/7.
